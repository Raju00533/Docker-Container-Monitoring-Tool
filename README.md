# Docker-Container-Monitoring-Tool
Docker Container Monitoring Tool 


### How to run This Tool.


```
#Set the value to the Docker host variable.

export DOCKER_HOST=unix:///var/run/docker.sock

python3 main.py

#by default it will run on the 8000 port. 

```
